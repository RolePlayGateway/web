The First Decentraland Experiment

Hi everybody!

The first experiment with Decentraland has been a success. We'd like to thank everyone who helped testing it! We gathered a lot of feedback while the app stressed our connections and CPUs.

We are working on improving performance and the network protocol. We'll release a new genesis block in a couple of weeks. We switched this live version to test mode, so you can still try it out.

We started a mailing list, decentraland-dev and a subreddit where we'll discuss the strategy and next steps.

Stay tuned!

The Decentraland Team
